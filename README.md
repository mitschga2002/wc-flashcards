# Flashcards Quiz

A flashcard quiz app built with [Lit](https://lit.dev/) web components and TypeScript.

## Features

- One question at a time with answer feedback
- Carousel card transitions & keyboard shortcuts (1–9, Enter)
- End-screen stats (score %, correct, wrong, total)
- Light & dark theme (auto via `prefers-color-scheme`)
- Responsive layout

## Getting Started

```bash
npm install
npm run dev
```
